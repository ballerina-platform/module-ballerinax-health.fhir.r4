## Overview

Belgium Federal Medication (1.0.0) FHIR R4 profile package for medication dispensing interoperability scenarios. It includes typed records aligned with Belgian medication profile constraints.

> **Trademark Notice:** FHIR® and the FHIR® logo are the registered trademarks of [Health Level Seven International](https://www.hl7.org/) and their use does not constitute endorsement by HL7®.

### Key Features

- Typed BeMedicationDispense model aligned to Belgian IG
- StructureDefinition-based resource representation for safer integrations
- Supports medication dispensing data exchange workflows
- Integrates with repository-wide FHIR tooling packages
- JSON-oriented FHIR R4 profile handling

Ballerina package containing FHIR resource data models
compliant with https://www.ehealth.fgov.be/standards/fhir/medication/ implementation guide.
